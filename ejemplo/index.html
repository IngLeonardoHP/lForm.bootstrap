lForm.bootstrap

Leonardo Hernández

Introducción

Plugins para el manejo de ajax en form de html5 para Bootstrap 3.x, ofrece validacion de campos input, textarea y select. Tambien un generador de password alfanumerico con la opcion de incluir caracteres especiales.

lForm.bootstrap maneja el estado del boton durante el envio de datos con opciones multiples para la visualización del usuario.

Se mejora la velocidad con la que el desarrollador codifica al momento de hacer validaciones

Proximamente incluirá en el ajax envio de imágenes

Descargar

Descargar la libreria aquí e incluya la carpeta en su proyecto.

Incluir la libreria

          <link rel="stylesheet" href="lForm.bootstrap/css/lForm.bootstrap.css">
          <script type="text/javascript" src="lForm.bootstrap/js/lForm.bootstrap.js"></script>

Palabras clave JS

          $.validarInput('#contenedor');
          //Retorna 0 o 1, Esta funcion recibe un contenedor donde se encuentran los input`s a evaluar, si alguno esta vacio, lo señaliza y retorna 0, en caso contrario retorna 1.
          $.validarTextarea('#contenedor');
          //Retorna 0 o 1, Esta funcion recibe un contenedor donde se encuentran los textarea`s a evaluar, si alguno esta vacio, lo señaliza y retorna 0, en caso contrario retorna 1.
          $.validarSelect('#contenedor');
          //Retorna 0 o 1, Esta funcion recibe un contenedor donde se encuentran los select`s a evaluar, si la opcion seleccionada tiene valor 0, lo señaliza y retorna 0, en caso contrario retorna 1.
          $.password(int,boolean);
          //Retorna una cadena de caracteres alfanumerico, ejemplo "Lsj25a9y!.@ada", ideal generar claves, el primer parametro es la longitud de la cadena y el segundo es opcional y hace referencia a si se quieren o no caracteres especiales.

          btn.cargando(string,boolean,boolean);
          //Inicializa el btn cargando, recibe 3 parametros, el primero el texto que aparece en el btn mientras esta cargando, el segundo si se desea desactivar el btn durante la carga y el ultimo si se deseea animar el btn.
          btn.restaurar();
          //Restaura el btn a sus valores iniciales.

          $.limpiarCampos(".panel-body");
          //Limpiar campos input del contenedor

Opciones HTML5

Ignorar un campo

          data-ignorar="true"
          Dentro del input que se desea ignorar, Ejemplo:
          <input class="form-control" placeholder="Este input sera ignorado" id="inputIgnorado" type="text" data-ignorar="true">

Longitud maxima y minima

          data-lengthMax="int"
          Dentro del input que se desea limitar, Ejemplo:
          <input class="form-control" placeholder="Este input sera limitado" id="inputIgnorado" type="text" data-lengthMax="12">

          data-lengthMin="int"
          Dentro del input que se desea limitar, Ejemplo:
          <input class="form-control" placeholder="Este input sera limitado" id="inputIgnorado" type="text" data-lengthMin="2">

funciones.js

$("#registrar").click(function(){
  var btn=$(this);
  if($.validarInput(".panel-body")){
      btn.cargando("Registrando",true,true);
      setTimeout(function(){
          btn.restaurar();
          $.limpiarCampos(".panel-body");
      },2000);
  }
});
