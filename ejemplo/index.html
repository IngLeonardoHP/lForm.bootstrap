lForm.bootstrap

Leonardo Hernández

Introducción

Plugins para el manejo de ajax en form de html5 para Bootstrap 3.x, ofrece validacion de campos input, textarea y select. Tambien un generador de password alfanumerico con la opcion de incluir caracteres especiales.

lForm.bootstrap maneja el estado del boton durante el envio de datos con opciones multiples para la visualización del usuario.

Se mejora la velocidad con la que el desarrollador codifica al momento de hacer validaciones

Proximamente incluirá en el ajax envio de imágenes

Key words JS

key Uso Ejemplo
$.validarInput(string) Retorna 0 o 1
Esta funcion recibe un contenedor donde se encuentran los input`s a evaluar, si alguno esta vacio, lo señaliza y retorna 0, en caso contrario retorna 1. 
            if($.validarInput('div')){
              console.log("No hay errores en los input's");
            }else{
              console.log("Error!");
            }ç
$.password(int,boolean) Retorna un string
Generar claves, ejemplo "Lsj25a9y!.@ada", el primer parametro es la longitud de la cadena y el segundo es opcional y hace referencia a si se quieren o no caracteres especiales. 
            alert($.password(12,true));
$.limpiarCampos(string) Limpiar campos input del contenedor
Recibe el contenedor de los input's que seran limpiados. 
            $.limpiarCampos("div");
$(button).cargando(string,boolean,boolean) Inicializa el btn cargando
Recibe 3 parametros, el primero es el texto que aparece en el button mientras este cargando, el segundo si se desea desactivar el button durante la carga y el ultimo si se deseea animar el button. 
            $("button").cargando("cargando",true,true);
$(button).restaurar(string,boolean,boolean) Restaura el btn a sus valores iniciales. 
            $("button").restaurar();
$.lFormAjaxMagic(string) Recibe el id de un formulario
Convierte un formulario normal de HTML5 en un ajax (Incluye envio de imagenes)

Nota importante: Debe tenerse cuidado con no olvidar agregar en el formulario el "METHOD" y "ACTION", como tambien el "name" en cada input

 
            $.lFormAjaxMagic('#form');

Opciones HTML5

Ignorar un campo

          data-ignorar="true"
          Dentro del input que se desea ignorar, Ejemplo:
          <input class="form-control" placeholder="Este input sera ignorado" id="inputIgnorado" type="text" data-ignorar="true">

Longitud maxima y minima

          data-lengthMax="int"
          Dentro del input que se desea limitar, Ejemplo:
          <input class="form-control" placeholder="Este input sera limitado" id="inputIgnorado" type="text" data-lengthMax="12">

          data-lengthMin="int"
          Dentro del input que se desea limitar, Ejemplo:
          <input class="form-control" placeholder="Este input sera limitado" id="inputIgnorado" type="text" data-lengthMin="2">

$("#registrar").click(function(){
  var btn=$(this);
  if($.validarInput(".panel-body")){
      btn.cargando("Registrando",true,true);
      setTimeout(function(){
          btn.restaurar();
          $.limpiarCampos(".panel-body");
      },2000);
  }
});
